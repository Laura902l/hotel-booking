# Hotel Booking 

Приложение для бронирования комнат в отеле.

## Функциональность

* Просмотр списка комнат
* Фильтрация и сортировка по:
  * цене
  * количеству мест
* Поиск свободных комнат в заданном интервале дат
* Бронирование комнаты
* Отмена бронирования
* Регистрация и авторизация пользователей
* Просмотр своих бронирований
* Администрирование через Django Admin


Дополнительно:

* автотесты
* линтер
* автоформатирование кода



## Запуск проекта

### 1. Клонировать репозиторий

```
git clone <repo_url>
cd hotel-booking
```



### 2. Запуск через Docker

```
docker compose up --build
```

После запуска приложение будет доступно:

```
http://localhost:8000
```


## API документация

Swagger UI доступен по адресу:

```
http://localhost:8000/api/docs/
```



## Админ панель

```
http://localhost:8000/admin/
```



## Автотесты

В проекте реализованы автотесты для проверки:

* получения списка комнат
* создания бронирования
* запрета пересекающихся броней
* отмены бронирования

Запуск тестов:

```
docker compose exec web python manage.py test
```



## Линтер и форматирование

В проекте используется линтер и автоформатирование кода.

Запуск линтера:

```
ruff check .
```

Форматирование:

```
ruff format .
```
